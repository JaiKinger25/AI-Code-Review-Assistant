# AI Code Review Assistant

## Overview

AI Code Review Assistant is a web application that analyzes Python source code using static analysis tools and provides AI-powered feedback.

---

## Features

- User Authentication (JWT)
- Upload Python (.py) Files
- AI Code Review
- Pylint Analysis
- Bandit Security Analysis
- Radon Complexity Analysis
- Automatic AI Fallback when external AI is unavailable

---

## Technologies Used

### Frontend
- React.js
- Axios
- CSS

### Backend
- Flask
- SQLAlchemy
- JWT Authentication

### Code Analysis Tools
- Pylint
- Bandit
- Radon

### AI
- Google Gemini API
- Local AI Review Engine (Fallback)

---

## Project Workflow

1. User logs in.
2. Uploads a Python file.
3. Backend stores the file.
4. Pylint analyzes code quality.
5. Bandit scans for security issues.
6. Radon calculates code complexity.
7. AI generates suggestions.
8. Results are displayed to the user.
